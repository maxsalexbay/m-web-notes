# m-web-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及不同主题，彼此之间没有强关联，均为独立编写的单页文件。仓库本身不针对任何特定域名或网站，仅作为页面资料的存储与展示空间。

## 目录说明

```
m-web-notes/
├── README.md          # 本文件
├── index.html         # 可选的仓库首页导航页
├── pages/             # 存放各类独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── assets/            # 公共资源文件（如 CSS、JS、图片等）
    ├── styles/
    ├── scripts/
    └── images/
```

- `pages/` 目录下存放所有独立 HTML 页面，每个文件为一个完整的页面。
- `assets/` 目录用于存放页面间可复用的静态资源，便于维护。

## 页面归档说明

仓库中每个 HTML 页面均为独立完整文件，可直接通过浏览器打开查看。页面内容可能随时间更新或替换，旧版本可通过 Git 历史记录追溯。建议在提交新页面或修改时，保持文件命名清晰，并在提交信息中简要描述变更内容。
